;;;;;====version 0.001========;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;
; 15/01/2020 Это скрипт на языке AutoIT разработанный чтобы брать
; и отказываться от квестов в сетевом клиенте Аллодов 2 (Rage of Mages)
; Требования:
; разрешение экрана: 1024x768
; Проверен на хэте Тангара (http://tangar.info/allods2)
;
; Лицензия: свободен к распространению, модификации, делайте с ним что угодно,
; но, пожалуйста, распространяйте вместе с исходным кодом.
;
; Этот скрипт предоставляется КАК ЕСТЬ. Используйте на свой страх и риск!
; Автор не несёт ответственности за результаты его работы.
; Вы можете потерять данные и вообще может случиться что угодно.
; Я же не заставлял вас его запускать :)
;
;
;   Как пользоваться
;  0) Скачать и установить AutoIt (https://www.autoitscript.com/site/autoit/downloads/)
;  1) Запустить клиент Allods2
;  2) Открыть QuestDeclining.au3 в редакторе AutoIt. Нажать "F5"  (команда "Go" для запуска скрипта). Скрипт переключится на окно запущенных Аллодов2.
;  3) Подойдите к таверне, наведите на неё курсор мыши и нажмите кнопку "/"
;  5) Подождите пока скрипт показывает доступные квесты. Если выбор не сделан, то он откажется от всех.
;  6) Нашли подходящий квест? Немедленно нажмите "\" и скрипт самоуничтожится :)
;  Проблемы? Нажмите "\" и скрипт самоуничтожится.
;  Файл журнала для отладки "a2q.log" записывается в каталог со скриптом
;  Файл настроек "a2q.log" записывается в каталог со скриптом. Редактируйте значения под ваши нужды.
