PhpFM версия 2.4 pro
================
Это совершенно новый подход к online-файловым менеджерам.
В формате .txt -  потому что HTML уже надоел.  К тому же в
текстовых файлах невозможно передать вирусы. И открывается
в любой программе по работе с текстом.


Основные преимущества:
================
- высокая скорость работы
- очень удобный и интуитивно понятный интерфейс
- практически все возможности обычного windows explorer -
  редактирование, переименование,  копирование и удаление
  файлов и папок.
- поддержка Gzip-сжатия
- возможность  скачать  нужный  вам файл с использованием
  Gzip-сжатия
- генерирование небольшого объема HTML-кода
- работает как в Windows так и в Unix.
- закачка директории целиком (new!)


Установка:
================
Установка в общем-то не требуется.  Единственное,  что
может понадобиться - сменить пароль. Редактируйте файл
check.pl  -  не  волнуйтесь,  в любом случае этот файл
будет  недоступен из Web  (если на вашем хостинге есть
Perl - чаще всего он идет вместе с PHP)

Права файлов:
*.php - 0755 (у разнх хостеров по-разному)
description, images - так, чтобы было доступно из web.
counters - 0666 (права на чтение и запись)

P.S. Чтобы файлы с неизвестным расширением отображались
правильно, откройте папку images и подправьте в .htaccess
путь до файла "images/file-.png" по отношению к браузеру.


Закачка директории :
================
Не обольщайтесь, для закачки директории потребуется  иметь
собственный web-сервер (если у вас его нету, посетите сайт
http://web.dklab.ru - несмотря на то,  что он весит  всего
2 мегабайта, в него входят полные версии Apache, PHP, Perl
и MySQL. Собственно, под  этим  сервером  и  делалась  эта
программа.)

Что нужно сделать после установки Денвера:
1) закачать в папку  /home/localhost/phpfm20 (создайте её)
весь файловый менеджер.
2) запустить сервера (Start Servers) 
3) зайти по адресу http://localhost/phpfm20

После того, как вы зашли в файловый менеджер:
1) заходите в папку, которую вы хотите закачать
2) нажимаете "дополнительные возможности"
3) находите меню "подготовка закачки директории"
4) указываете "имя папки" - "." (точка)
5) размер фрагментов можете не трогать
6) дальше следуйте инструкциям phpFM

!!! Внимание - закачиваются только файлы без подпапок !!!


История:
================
Версия 2.4
	+ добавлена защита от открытия слишком  больших файлов
	в режиме редактирования
	+ в очередной раз немного изменен интерфейс
	+ добавлена панель «подробно»
	+ добавлена поддержка клавиатуры!!!:
	  1) стрелочками можно выбирать файлы вверх-вниз
	  2) по нажатию клавиши Enter папка открывается, либо
	     открывается файл для редактирования
	  3) по нажатию Alt+Enter выскакивает окно свойств
	     папки или файла
	  4) по нажатию на Backspace (клавиша «стереть») вы
	     перемещаетесь на 1 каталог вверх
	  5) по нажатию на DELETE вам предлагается удалить
	     файл или папку
	  6) по нажатию на F2 вам предлагается переименовать
	     файл или папку 
Версия 2.2  - есть поддержка пользователей !
	+ добавлена поддержка пользователей
	+ добавлен config-editor (online-редактор конфигов)
	+ в очередной раз улучшено контекстное меню
	+ в поддержке пользователей:
	  1) есть фильтр всех исполняемых и загружаемых файлов
	     6 степеней защиты
	  2) удобное меню администратора
	  3) в меню администратора встроенная Справка
	  4) есть дисковые квоты для пользователей
	  5) продуманная база данных  пользователей - забудьте
	     о конверторах, база будет переделываться сама при
	     редактировании базы  пользователей  (для  будущих
	     версий)
	  6) защита от взлома базы -  даже  если  пользователь
	     имеет доступ к папке с phpfm, он ничего не сможет
	     с ней сделать (правда, исполнение PHP-кода  может
	     убить всю эту защиту)
	+ улучшена защита сессии администратора
	+ защита от слишком умных пользователей - пользователи
	  не смогут скопировать, удалить, переименовать файлы,
	  находящиеся за пределами их папки
	+ добавлен режим "эскизы страниц" !

	+ Немного изменен интерфейс (в лучшую сторону)
